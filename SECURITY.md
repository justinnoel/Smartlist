# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 4.0     | :white_check_mark: |
| 3.0     | :x:                |
| 2.0     | :x:                |
| 1.0     | :x:                |
| < 1.0   | :x:                |

### Security specs

* Argon2 Hashing
* AES-256 encryption
* Prepared SQL statements
* SQL and XSS proof
* Removes html characters
* Uses HTTP/3
* Uses Cloudflare
* HSTS enabled
